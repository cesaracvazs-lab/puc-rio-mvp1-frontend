# Gerenciador de Assinaturas - Front-end

Este projeto corresponde ao front-end da aplicação de gerenciamento de assinaturas desenvolvida na disciplina **Desenvolvimento Full Stack Básico**.

A interface foi construída com **HTML**, **CSS** e **JavaScript puro**, consumindo uma API HTTP local para listar clientes, exibir detalhes, editar dados e consultar assinaturas.

---

## Funcionalidades

- Listagem de clientes em tabela
- Visualização detalhada de um cliente
- Edição de dados cadastrais do cliente
- Exclusão de clientes
- Consulta de assinaturas vinculadas

---

## Pré-requisitos

Antes de executar o projeto, garanta que você tenha:

- O back-end da aplicação em execução localmente
- A API disponível no endereço `http://127.0.0.1:5000/`

Observação: este front-end faz requisições para a API local definida em `src/App.js`.

---

## Instalação

Como este projeto é estático, não há dependências de `npm` para instalar.

1. Faça o download ou clone este repositório https://github.com/cesaracvazs-lab/puc-rio-mvp1-frontend.git
2. Acesse a pasta do projeto front-end.
3. Certifique-se de que o back-end esteja iniciado e respondendo em `http://127.0.0.1:5000/`.

Exemplo de clonagem:

git clone <url-do-repositorio>
cd front-end

---

## Como executar

1. Localize o arquivo `index.html`.
2. Abra o arquivo no navegador de sua preferência.
